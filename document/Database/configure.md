### 전체 테이블 리스트 보기
```
show databases;
```

### 데이터베이스 선택
```
use database_name;
```

### 테이블 리스트 보기
```
show tables;
```

### 데이터베이스 비우기
- 모든 테이블을 하나씩 drop 명령어로 삭제한다.
```
drop table table_name1;
drop table table_name2;
drop table table_name3;
```
```
drop table table_name1, table_name2, table_name3;
```
