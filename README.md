## make laravel project
```
composer create-project --prefer-dist laravel/laravel projectnName
```

## init setting
```
composer install
composer update
```

## install authenticate scaffolding
```
composer require laravel/ui --dev
php artisan ui:auth bootstrap
npm install
```


## nodejs build
```
npm run watch
npm run production
```
> error
```
npm install -f
```
